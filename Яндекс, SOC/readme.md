# Анализ инцидента безопасности на устройстве сотрудника johndoe

## ИСПОЛНИТЕЛЬНОЕ РЕЗЮМЕ
На устройстве сотрудника был выявлен инцидент безопасности, связанный с компрометацией через фишинговую атаку. Злоумышленник использовал социальную инженерию для получения доступа, после чего развернул вредоносное ПО для установления постоянного доступа и эксфильтрации данных.

- 18:05:28 - Запуск вредоносного файла Invoice.pdf.exe

- 18:26:42 - Повторная активация вредоносного процесса
- 18:27:42 - Установление сетевых соединений с C&C-сервером

## ОТВЕТЫ НА ВОПРОСЫ:

### 1. Как скомпроментировали устройство сотрудника?
**Вредоносный файл:** Invoice.pdf.exe  
**Способ получения:** Фишинговая атака через email с архивом "Invoice.zip", содержащим вредоносный исполняемый файл под видом PDF-документа.

### 2. Какие инструменты использовал злоумышленник? Что пытался сделать?
- PowerSploit.psd1 - фреймворк пост-эксплуатации для выполнения различных атак
- winPEAS.bat - инструмент перечисления системы для поиска уязвимостей и сбора информации
- Invoke-DllInjection - внедрение вредоносной DLL в легитимные процессы
**Цель:** получение постоянного доступа, сбор информации о системе, подготовка к эксфильтрации данных.

### 3. IP-адрес C&C-сервера злоумышленника:
`212.118.39.217:9090`

### 4. Как злоумышленник обошел Windows Defender?
Использовал команду PowerShell: `Set-MpPreference -DisableRealtimeMonitoring $true`  
Это отключило реальное время мониторинга антивируса.

### 5. Как злоумышленник обошел AMSI?
Использовал команду `Set-StrictMode -Off` в PowerShell, что является распространенной техникой обхода системы антивирусного мониторинга скриптов (AMSI).

## РЕКОМЕНДАЦИИ ПО РЕАГИРОВАНИЮ:
1. Изолировать зараженное устройство из сети
2. Выполнить полное сканирование системы антивирусным ПО
3. Сменить все учетные данные сотрудника
4. Провести аудит сетевой активности за последние дни
5. Усилить обучение сотрудников по распознаванию фишинговых атак
6. Рассмотреть внедрение дополнительных мер безопасности (MFA, DLP-системы)

## ЗАКЛЮЧЕНИЕ:
Инцидент классифицируется как целевая фишинговая атака с элементами APT (Advanced Persistent Threat). Злоумышленник продемонстрировал хорошее знание инструментов пентеста и техник обхода защитных механизмов Windows. Рекомендуется провести дополнительное расследование для выявления возможного ущерба от эксфильтрации данных.
