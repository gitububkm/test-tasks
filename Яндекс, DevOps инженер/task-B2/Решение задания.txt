чтобы сохранить изменения без коммита используем временное хранилище
git stash

переходим в другую ветку
git checkout main

потом чтобы вернуться в изначальную ветку
git checkout feature/junior-task

и возвращаем сохраненные изменения из временного хранилища
git stash pop

и чтобы переименовать последний коммит 
git commit --amend -m "NEW COMMIT NAME"
